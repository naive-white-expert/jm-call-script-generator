## 一、核心演进：从"替换"到"生成"

### 1.1 常见错误类型

#### 错误1：部分替换导致不通顺

**错误示例**：
```
原句：我们给您京东账户赠送了一张50元专属优惠券
         ↓ 机械替换（只替换部分词汇）
结果：我们给您京东账户赠送了的京东  ❌ 不通顺
```

**原因**：删除"专属优惠券"后，句子结构被破坏

**正确做法**：完整句子替换
```
原句：我们给您京东账户赠送了一张50元专属优惠券，券后到手低至83元
         ↓ 完整替换
结果：我们给您京东账户准备了5元直减红包，活动到手垃圾袋9.9元  ✅
```

---

#### 错误2：机械拼接信息块

**错误示例**：
```
开场白 = 开场语 + 钩子 + 引导
       = "喂您好，这边是京东的，您好？" + "就是送您京东专属福利" + "我给您简单说下吧？"
结果：喂您好，这边是京东的，您好？，就是送您京东专属福利，我给您简单说下吧？  ❌
```

**原因**：机械拼接，标点符号错误，节奏混乱

**正确做法**：保持原模板的表达节奏

---

#### 错误3：删除内容后句子残缺

**错误示例**：
```
原句：就是您账户里已经到账一张50元专属券，用券后到手低至83元
删除"专属券"后：就是您账户里已经到账一张50元，用活动到手低至83元  ❌
```

**问题**：
- "到账一张50元"缺少对象
- "用活动到手"表达不通

**正确做法**：重新组织句子结构

---

#### 错误4：重复表达

**错误示例**：
```
原句：券后到手低至83元
替换后：券后到手买4袋叠券后476元  ❌ 重复"券后"
```

**原因**：benefit_detail中包含了"券后"，导致重复

**正确做法**：benefit_detail只保留价格，避免重复

---

#### 错误5：原模板残留

**错误示例**：
```
场景一（非宠物）：还有正装宠物香氛  ❌ 
场景二：还有网易严选紫金烘焙猫粮  ❌
```

**原因**：未完全替换原模板内容

**正确做法**：系统性校验，删除所有原模板残留

## 二、替换规则详细示例

### 2.1 完整句子替换规则

**核心规则**：识别原模板中的完整句子，根据场景信息重新生成整句，不进行部分替换

**替换优先级**：长词组优先（避免多次替换导致不通顺）

---

### 2.2 场景类型处理示例

#### 示例1：有专属券场景

**原句**：
```
我们给您京东账户赠送了一张50元专属优惠券，券后到手低至83元，还有猫条等9件好礼
```

**替换后**：
```
我们给您京东账户赠送了一张无门槛8折券，券后到手猫粮3.0 271.2元，还有299减45红包
```

**处理方式**：
- 保留"券"的表达
- 替换券的类型和金额
- 替换价格和赠品信息

---

#### 示例2：无专属券场景

**原句**：
```
就是您账户里已经到账一张50元专属券，用券后到手低至83元，还额外赠送9件好礼
```

**替换后**：
```
就是我们给您准备了5元直减红包，活动到手垃圾袋9.9元，T恤94元3件，还有10周年大促券200减30
```

**处理方式**：
- 删除"券"的表达
- 重新组织句子结构：账户到账券 → 准备红包
- 替换价格表达：用券后价格 → 活动价格

---

#### 示例3：非宠物行业

**原句**：
```
就是送您网易严选紫金烘焙猫粮专属福利，趁活动优惠快给咱家毛孩子囤点好吃的吧
```

**替换后**：
```
就是送您京东专属福利，趁活动优惠快给咱家家人囤点好吃的吧
```

**处理方式**：
- 替换品牌：网易严选紫金烘焙猫粮 → 京东
- 替换行业词汇：毛孩子 → 家人

---

#### 示例4：犬粮产品

**原句**：
```
我们给您京东账户赠送了一张50元专属优惠券，用券下单猫粮低至83元每袋，趁活动优惠快给咱家毛孩子囤点好吃的吧
```

**替换后**：
```
我们给您京东账户赠送了一张叠198减30元老客券，券后到手262元，趁活动优惠快给咱家狗狗囤点好吃的吧
```

**处理方式**：
- 替换产品词汇：猫粮 → 犬粮
- 替换称呼：毛孩子 → 咱家狗狗

---

### 2.3 长词组优先示例

**正确做法**：长词组优先替换
```
原句：网易严选京东自营旗舰店
正确：一次替换为"网易天成京东自营旗舰店" ✅
错误：先替换"网易严选"→"网易天成"，再替换"京东自营旗舰店"，导致"网易天成京东自营旗舰店京东自营旗舰店" ❌
```

## 七、口语化表达规则

### 7.1 核心原则

**电话话术 ≠ 书面文案**

电话沟通的特点：
- 用户在听，不是在看
- 需要实时理解和记忆
- 信息量过大会遗漏
- 需要自然的停顿和转折

**核心要求**：
- ✅ 口语化表达，像真人说话
- ✅ 单一权益一句话
- ✅ 信息分块，有停顿节奏
- ❌ 避免书面化堆砌
- ❌ 避免信息过载

---

### 7.2 权益表达对比

#### 7.2.1 优惠券/红包表达

| 书面化表达 | 口语化表达 | 说明 |
|-----------|-----------|------|
| ❌ "赠送了50元优惠券" | ✅ "送了一张50元的优惠券" | 加量词"一张" |
| ❌ "5元直减红包" | ✅ "一张5元直减红包" | 加量词"一张" |
| ❌ "无门槛8折券" | ✅ "一张无门槛8折券" | 加量词"一张" |
| ❌ "给您账户赠送了XX" | ✅ "给您账户送了一张XX" | "送了"更口语 |

#### 7.2.2 使用引导表达

| 书面化表达 | 口语化表达 | 说明 |
|-----------|-----------|------|
| ❌ "XX产品红包后9.9元" | ✅ "用这个红包，XX产品到手只要9块9" | 加引导"用这个" |
| ❌ "券后价格83元" | ✅ "用券后到手只要83块钱" | 加"到手"更自然 |
| ❌ "低至83元" | ✅ "折合下来只要83块" | 加"折合下来" |

#### 7.2.3 时间表达

| 书面化表达 | 口语化表达 | 说明 |
|-----------|-----------|------|
| ❌ "有效期4.6-4.11" | ✅ "有效期到4月11号" | 日期转口语 |
| ❌ "活动时间X.X-X.X" | ✅ "这几天都能用" | 加使用说明 |

#### 7.2.4 价格表达

| 书面化表达 | 口语化表达 | 说明 |
|-----------|-----------|------|
| ❌ "9.9元" | ✅ "9块9" | 数字转口语 |
| ❌ "83元" | ✅ "83块钱" | 加"块钱" |
| ❌ "红包后XX元" | ✅ "到手只要XX块X" | 加"到手" |

### 7.4 句子节奏规则

#### 7.4.1 信息分块原则

**核心规则**：单一权益一句话，不要堆砌

**错误示例（堆砌）**：
```
❌ "来电呢是我们给您京东账户赠送了5元红包和200减30大促券，
京东京造10周年大促，厨房抽绳垃圾袋红包后只要9.9元，
纯棉抗菌纯棉T恤3件红包后94元"
```

**问题分析**：
- 多个权益塞在一句话
- 没有停顿和转折
- 用户难以消化

**正确示例（分块）**：
```
✅ "来电呢是给您账户送了一张5元直减红包，有效期到4月11号。
[停顿]
用这个红包，咱们厨房专用的特厚抽绳垃圾袋，3卷60只到手只要9块9。
[停顿]
还有10周年大促券200减30，跟红包一起用能减35。
[停顿]
像10A抗菌纯棉T恤，任选3件只要94块钱。
[收尾]
诚邀您查收下活动短信来享受优惠呀。"
```

#### 7.4.2 信息递进结构

| 步骤 | 内容 | 示例 | 作用 |
|------|------|------|------|
| 第1句 | 核心权益 | "送了一张5元直减红包，有效期到4月11号" | 抓住注意力 |
| 第2句 | 使用引导+产品价格 | "用这个红包，垃圾袋到手只要9块9" | 告诉怎么用 |
| 第3句 | 次要权益+叠加说明 | "还有200减30大促券，跟红包一起用能减35" | 增加吸引力 |
| 第4句 | 其他产品价格 | "像T恤，任选3件只要94块钱" | 扩大选择 |
| 第5句 | 行动号召 | "诚邀您查收下活动短信来享受优惠呀" | 引导参与 |

#### 7.4.3 停顿位置规则

**必须停顿的位置**：

1. **权益说完后**：
   ```
   ✅ "送了一张5元红包，有效期到4月11号。[停顿]"
   ```

2. **产品价格说完后**：
   ```
   ✅ "垃圾袋到手只要9块9。[停顿]"
   ```

3. **权益转换时**：
   ```
   ✅ "还有10周年大促券...[停顿]"
   ```

4. **举例产品前**：
   ```
   ✅ "像XX产品...[停顿]"
   ```

**禁止连续说**：
```
❌ "送了XX券XX产品XX元还有XX券XX产品XX元"
```

**错误示例（书面化堆砌）**：

纯棉抗菌纯棉T恤3件红包后94元，诚邀您查收下活动短信来享受优惠呀。"
```

**问题**：
1. "赠送了"太书面
2. 权益信息堆砌
3. 缺少使用引导
4. 没有停顿节奏

**正确示例（口语化分块）**：

像10A抗菌纯棉T恤，任选3件只要94块钱。
诚邀您查收下活动短信来享受优惠呀。"
```

**改进点**：
1. ✅ "送了一张"更口语
2. ✅ 单一权益一句话
3. ✅ "用这个红包"引导使用
4. ✅ 有停顿节奏

### 7.6 口语化检查清单

在生成话术后，检查以下要点：

**检查项1：量词使用**
- [ ] 优惠券/红包是否加"一张/一个"
- [ ] 价格是否用"XX块X"

**检查项2：连接词使用**
- [ ] 是否有"送了一张"而非"赠送了"
- [ ] 是否有"用这个XX"引导使用
- [ ] 是否有"还有"、"像"等连接词

**检查项3：信息分块**
- [ ] 是否单一权益一句话
- [ ] 是否有停顿节奏
- [ ] 是否避免堆砌

**检查项4：表达口语化**
- [ ] 时间是否用"X月X号"而非"X.X-X.X"
- [ ] 价格是否用"到手只要XX"而非"红包后XX元"
- [ ] 是否避免书面化表达
